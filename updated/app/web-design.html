Hello, My Name is

Jacob Perry

Web Designer

Programming was something that I was always interested in but never really thought to seriously about as a child. I thought that is was something that people much smarter than me would do and I would just work on doing art. But as it turns out I wasn’t all that great at art either so I started to think more seriously about programming. So in high school I took an intro to web design class and learned the basics of HTML and CSS and was pretty happy with the results but as I tried to learn more about web programming I just overwhelmed with all the buzzwords and frameworks and tools that I just didn’t know where to start. It wasn’t till I had decided to learn how to program in C# making games that I thought I would give web design another shot, but this time I chose to take an online classes. It was in these online classes that I started to learn to enjoy web programming as much as I had learned to love making games. I finally understood what it took to make a website that worked and looked good and was more than just some HTML and CSS.
